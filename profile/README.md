## 🚀 Projects

### 🌿 [IoT Green Panels](https://github.com/HardCounter/BHL2025) — BHL 2025 Winner
Self-sustaining, modular hexagonal panels with living moss that purifies air, sun-tracking solar power, automated irrigation, integrated fire suppression, and a real-time React + Python web dashboard.

### 🕹️ [Interactive Gaming Robot](https://github.com/HardCounter/BiteHack2026) — BiteHack 2026 Winner
A mobile multiplayer gaming platform with dual remote controllers, co-op driving mode, and an onboard arcade mini-games library.

### 🧤 [PowerGlove](https://github.com/HardCounter/BHL2024) — BHL 2024 (2nd Place)
A wearable hub for astronauts featuring biometric monitoring, environmental sensing, spatial awareness with a laser rangefinder, and an auto-trigger emergency system.

### ✈️ [The Whimsy Wombat](https://github.com/HardCounter/The_Whimsy_Wombat)
A student-built remote-controlled plane with Arduino-based controls and a roadmap toward full autonomous flight via GPS and IMU stabilisation.
